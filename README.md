# Holiday-Postcard-Generator
A simple cross-platform desktop application built using the Xojo framework to quickly generate print-ready 4" x 6" PDF holiday postcards from a user-selected image and custom message.

## ğŸš€ Usage
The application provides a straightforward interface:

1.  **Select Image:**  Click the  **"Select Image"**  button. The application recommends using an image with a 4:6 aspect ratio (portrait orientation) for best results, such as 1200x1800 pixels.
2.  **Enter Message:**  Use the text area provided to type your holiday message. A default message is provided, which you can edit or replace entirely.
3.  **Generate PDF:**  Click the  **"Generate PDF"**  button.
4.  **Review:**  The application will save the resulting PDF to your desktop and attempt to open it using your system's default PDF viewer. The status bar at the bottom will confirm success or display any errors encountered.
